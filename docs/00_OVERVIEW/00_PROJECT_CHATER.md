# プロジェクト名: Cocoba (ここちゃん × ルンバ)

## 1. 背景と課題
* **対象:** 14歳のシニア犬（ここちゃん）
* **背景:** 日中留守中の排泄物を犬が踏み散らかし、床が広範に汚れる。帰宅後の固まった排泄物の清掃に毎日30分を要しており、飼い主の負担が大きい。
* **制約事項:** シニア犬のため、新規のトイレトレーニングは不可。また、稼働時の接触による怪我を防ぐため、安全確保が最優先される。

## 2. プロジェクトの目的
排泄物踏み散らかしによる二次災害を完全に防止し、運用者（飼い主）の日々の清掃コストを「毎日数分レベル」へ劇的に削減する。
同時に、Webシステム連携により外出先からの遠隔監視を可能にし、留守中の心理的負担（不安感）を取り除く。

## 3. 解決アプローチ
技術的ハードルの高い「物理的な回収機構」をスコープから除外する。
代わりに、以下の3本柱でシステムを構築する。
1. **隔離・シールド:** 排泄物を物理的なドームで「覆う」ことに特化し、犬のアクセスを遮断する（エッジAIとロボットの連携）。
2. **誘引・退避:** 安全な別領域へ少量の餌を排出し、犬を排泄箇所から物理的に遠ざける。
3. **遠隔介入 (Web連携):** システムの状態をWebアプリケーションへ同期させ、外出先からのステータス確認と、緊急時の手動停止を可能にする。

## 4. 成功定義 (KPI)
* 犬の排泄物踏み付け事故: 2件／月
* 帰宅後の清掃時間: 5分／件
* ロボットと犬の接触・衝突事故: 0件／月
* 異常発生時の飼い主への通知遅延: 10秒以内
