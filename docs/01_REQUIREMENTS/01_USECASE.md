# 運用ユースケース

## 1. 運用の基本原則
* **ゼロ・タッチ・オペレーション:** 日中の無人環境下において、飼い主の能動的な介入（Web画面に張り付いて監視するなど）を一切不要とする。通知は「事後報告」または「異常発生時」のみに限定する。
* **物理タスクの極小化:** 出発前および帰宅後の物理的なセットアップ（弾込め、清掃）を簡単にするハードウェア設計・配置を前提とする。

## 2. 運用ユースケース 
Webシステム連携を含めた、実運用における代表的なシナリオとシステムの振る舞い。

* **UC1: 正常稼働と事後確認**
  * **状況:** 留守中にここちゃんが排泄。
  * **システム:** AI検知 → おやつ誘導 → ロボット出動・投下 → 帰還を完全自律で完了。
  * **Web/通知:** 投下完了後、飼い主のスマホへ「シールド完了」のLINE通知と、検知時のスナップショット画像を送信。
  * **飼い主の行動:** 通知を見て安心し、そのまま業務・外出を継続する。

* **UC2: 遠隔キルスイッチの発動 (緊急介入)**
  * **状況:** AIが排泄物を検知してロボットが出動したが、スナップショット画像を見た飼い主が「ただの黒い靴下（誤検知）」であることに気づいた。あるいは、ロボットの挙動がおかしいことに気づいた。
  * **システム:** 稼働中ステータスをWebへ同期中。
  * **飼い主の行動:** Webダッシュボードを開き、即座に `[緊急停止]` をタップ。
  * **結果:** エッジPCがコマンドを即時受信し、ロボットのモーターへの電力供給を論理的にカット。誤動作による事故を未然に防ぐ。

* **UC3: 誘導失敗によるフェイルセーフ発動**
  * **状況:** 排泄後、ここちゃんがおやつに反応せず、排泄物のすぐ横で寝転がってしまった。
  * **システム:** エッジPCが「犬が安全圏に退避していない（タイムアウト）」と判断。ロボットの出動をキャンセル。
  * **Web/通知:** 飼い主へ「誘導失敗：出動をキャンセルしました」のアラートを送信。二次被害（踏み散らかし）のリスクがあることを事前に伝える。

## 3. 業務フロー
飼い主が毎日必ず実行する物理的・論理的なアクション。このフローが守れない日は、システムを「稼働させない（電源OFF）」にすること。

### フェーズ1: 出発前
**【目標所要時間: 1分以内】**
1. **シールドの装填:** 洗浄・乾燥済みのドームを、Cocobaロボットのディスペンサーにスタック（最大積載数）する。
2. **誘引餌の補充:** おやつシューター（ESP32機）のサイロに、1日分のカリカリ（数粒×想定回数分）を投入する。
3. **ヘルスチェック (Web確認):** スマホでダッシュボードを開き、以下の「オールグリーン」を確認する。
   * エッジPCのステータスが `[待機中]` であること。
   * ロボットのドーム残弾数が正しく反映されていること。
   * カメラの最新映像に死角がないこと（カメラの向きズレ確認）。
4. **外出:** そのまま家を出る。

### フェーズ2: 日中
**【目標所要時間: 0分 (通知確認のみ)】**
1. システムからのPush通知（LINE等）を受信した際のみ、内容（スナップショット画像）を確認する。
2. 異常（誤検知や危険状態）を視認した場合のみ、Webダッシュボードから介入（キルスイッチ）を行う。

### フェーズ3: 帰宅後
**【目標所要時間: 3分以内】**
1. **システムの論理停止:** 帰宅時の飼い主の足などを誤検知しないよう、帰宅直後にWebダッシュボードから `[システムスリープ]` を実行し、監視を一時停止する。
2. **物理回収:** 床に配置されているドームを確認。ドームの上からトイレットペーパー等を被せ、中の排泄物ごと持ち上げてトイレ等へ廃棄する。
3. **清掃とリセット:**
   * 使用済みドームの裏面（フチ）をアルコール除菌シート等で拭き取る、または水洗いする。
   * Cocobaロボットを所定の待機位置（充電ケーブル接続）に戻す。

## 4. 定期保守
システムの稼働率を100%に保つための、週末等を活用したメンテナンス作業。

* **[週次] センサー・カメラ清掃:** ネットワークカメラのレンズ、およびロボットに貼付したArUcoマーカーのホコリを拭き取る（認識率低下の防止）。
* **[週次] おやつシューター清掃:** カリカリの油分による「弾詰まり」を防ぐため、排出機構（筒やモーターの羽）を清掃する。
* **[月次] ロボットの足回り点検:** ルンバ（または自作ローバー）のタイヤやキャスターに、犬の毛やホコリが絡まっていないか確認・除去する。
* **[月次] エッジPC再起動:** メモリリークやOSの隠れた不具合をリセットするため、ミニPCを計画的に再起動（リブート）する。
