# 04_HARDWARE_WIRING (ハードウェア配線設計)

## 1. 概要
本ドキュメントは、エッジPC、マイコン（ESP32）、モーター、および電源系統の物理的な結線（ピンアサイン）を定義する。
「電源の分離」と「物理キルスイッチの確実性」を回路レベルで保証する。

## 2. 電源系統 (Power Distribution)

ノイズと電圧降下（Brownout）を防ぐため、ロジック系（頭脳）と動力系（筋肉）を分離する。

*   **ロジック系電源 (5V):**
    *   モバイルバッテリー（パススルー充電対応）からUSB給電でESP32に接続。
    *   これにより、メイン電源断時もESP32の通信は維持され、クラウドへ「電源ロスト」の遺言（LWT）を残せる。
*   **動力系電源 (12V or 7.4V Li-Po):**
    *   ベースロボットの駆動モーター、およびサーボモーター用の大電流を供給。
*   **GNDの共通化:**
    *   ロジック系と動力系のGNDは、モータードライバのGNDピンで1点アース（スター配線）として接続し、信号の基準電位を合わせる。

## 3. ESP32 ピンアサイン (Pinout)

ESP32-DevKitC（38ピン）を想定した割り当て。

| ピン番号 | 役割 | 接続先 | 備考 |
| :--- | :--- | :--- | :--- |
| **GPIO 32** | 割り込み (INPUT_PULLUP) | 物理キルスイッチ | 押下時 GND に短絡。ISRで即時モーター停止。 |
| **GPIO 25** | PWM (出力) | 左モータードライバ IN1 | 前進用PWM |
| **GPIO 26** | PWM (出力) | 左モータードライバ IN2 | 後退用PWM |
| **GPIO 27** | PWM (出力) | 右モータードライバ IN3 | 前進用PWM |
| **GPIO 14** | PWM (出力) | 右モータードライバ IN4 | 後退用PWM |
| **GPIO 12** | PWM (出力) | サーボモーター 1 | シールド投下（右爪） |
| **GPIO 13** | PWM (出力) | サーボモーター 2 | シールド投下（左爪） |
| **3V3** | 電源 | 各種センサー (必要時) | - |
| **GND** | グラウンド | バッテリー・ドライバ | - |

## 4. 物理キルスイッチ回路

単なるソフトウェア処理に依存せず、**ハードウェア的にもモーターを止める**設計。

1.  **論理停止:** キルスイッチを GPIO 32 に接続し、割り込み（ISR）で ESP32 の全PWM出力を `0` にする。
2.  **物理遮断 (推奨):**
    *   動力系バッテリーのプラス（+）ラインの途中に、大電流対応の「ノーマルクローズ（B接点）のリレー」または「モーメンタリの物理スイッチ（キノコ型非常停止ボタン）」を直列に挟む。
    *   ボタンが押されると、モータードライバとサーボへの電源供給自体が物理的に遮断され、ソフトウェアが暴走していても確実に停止する。
