# 03_MECHANICAL (メカニカル・機構設計)

## 1. 概要
本ドキュメントは、Cocobaロボットの「シールド投下機構」および「おやつ排出機構（シューター）」の物理的な動作原理を定義する。
複雑なロボットアームを避け、重力を利用したフェイルセーフな機構を採用する。

## 2. シールド（ドーム）投下機構

### 2.1. 機構のコンセプト
*   **ディスペンサー方式:** 紙コップのディスペンサーのように、複数のドームをスタック（重ね置き）して上部に搭載する。
*   **重力落下:** サーボモーターは「一番下のドームを支えている爪」を引っ込めるだけの役割を担う。ドームは自重で落下し、目標地点に覆いかぶさる。
*   **利点:** モーターのトルク不足で押し込めない、または犬の体にアームが接触して怪我をさせるリスクが物理的に存在しない。

### 2.2. サーボ制御と状態
*   **使用アクチュエータ:** 金属ギヤ標準サーボ (例: MG996R) x 2個
*   **制御角:**
    *   `HOLD (0度)`: 爪が飛び出し、一番下のドームのフチを支えている状態。
    *   `RELEASE (90度)`: 爪が引っ込み、ドームが落下する。
*   **シーケンス:**
    1.  目標地点に到達後、2つのサーボを同時に `RELEASE` へ。
    2.  0.5秒待機（落下完了を待つ）。
    3.  ロボットを少し後退させ、次のドームが下まで降りてくる空間を作る。
    4.  サーボを `HOLD` に戻し、次のドームをキャッチする。

## 3. おやつ排出機構 (シューター)

### 3.1. 機構のコンセプト
*   **ロータリー方式:** 水車のような回転体に少量のカリカリ（ドッグフード）を入れ、回転させることで下部から排出する。
*   **利点:** スクリュー式に見られる「フードの詰まり（ジャム）」によるモーターのロックや過電流アウトを防止しやすい。

### 3.2. モーター制御
*   **使用アクチュエータ:** ステッピングモーター (例: 28BYJ-48) または連続回転サーボ。
*   **制御:**
    *   MQTTで `{"rotations": 1}` を受信。
    *   モーターを1回転（360度）駆動させ、規定量（約3〜5粒）を確実に排出する。
