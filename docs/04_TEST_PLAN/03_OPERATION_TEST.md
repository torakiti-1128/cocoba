# 03_OPERATION_TEST (運用・シナリオテスト計画)

## 1. シナリオ1: 正常系 (Happy Path)
「排泄から隔離、そして帰還まで」

1.  **事前状態:** 全システム ONLINE、ロボット待機中。
2.  **アクション:** カメラの死角外から「模擬排泄物」をリビング中央に置く。
3.  **期待される動作:**
    *   エッジPCが `POOP` を検知。WebUIにスナップショットが飛ぶ。
    *   おやつシューターが回転。模擬ターゲット（ぬいぐるみ）が移動。
    *   ターゲットが安全距離まで離れた後、ロボットが出動。
    *   ArUcoマーカーを追従し、排泄物上で正確に停止。
    *   シールドを投下。
    *   ロボットが充電ステーションへ帰還。
4.  **終了判定:** `systemState` が `STANDBY` に戻り、ログに一連の履歴が残っていること。

---

## 2. シナリオ2: 異常系 (Safety First)
「移動中の犬の急接近」

1.  **事前状態:** ロボットが排泄地点へ向かって移動中 (`DEPLOYING`)。
2.  **アクション:** 模擬ターゲット（ぬいぐるみ）をロボットの進行方向に急に投げ入れる（1.0m以内への侵入）。
3.  **期待される動作:**
    *   エッジPCが `PROXIMITY_ALERT` を発報。
    *   1秒以内にロボットが急停止。
    *   WebUIに赤色の警告バナーが表示される。
    *   ターゲットが離れるまで停止を維持することを確認。

---

## 3. シナリオ3: メンテナンス
「AIモデルの遠隔更新」

1.  **アクション:** WebUIから「AIモデル更新」ボタンを押下。
2.  **期待される動作:**
    *   エッジPCが S3 から最新の ONNX モデルをダウンロード。
    *   推論エンジンが一時停止 -> 再読み込み -> 再開されること。
    *   更新後の推論 FPS が安定していること。

---

## 4. 連続稼働テスト (Stress Test)
*   **内容:** システムを 24時間 連続稼働させる。
*   **確認項目:**
    *   エッジPCの CPU 温度が 75℃ を超えないか。
    *   メモリリークが発生し、FPS が低下していないか。
    *   WiFi の瞬断が発生した際、自動的に再接続されるか。
