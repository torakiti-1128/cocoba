# 04_NON_FUNCTIONAL_TEST (非機能テスト計画)

## 1. 安定性・連続稼働テスト (Reliability)
*   **72時間連続稼働テスト:**
    *   **内容:** エッジPC、カメラ、ESP32を 72時間 連続で通電・推論・通信させ続ける。
    *   **計測項目:** メモリ使用量の増加推移（リーク確認）、FPSの低下有無。
    *   **判定基準:** 72時間後に再起動なしで初期と同等のパフォーマンス（10 FPS）を維持していること。

## 2. 熱設計・負荷テスト (Thermal)
*   **高室温環境再現:**
    *   **内容:** エッジPC（N100）付近の温度を 35℃（夏季想定）に維持し、高負荷（YOLO推論 + クラウド通信）をかける。
    *   **計測項目:** CPU温度、サーマルスロットリングによるクロック低下回数。
    *   **判定基準:** CPU温度が 75℃ 以下に収まり、スロットリングによる推論の遅延（1fps以下への低下）が発生しないこと。

## 3. セキュリティテスト (Security)
*   **不正アクセス試行:**
    *   **内容:** 外部ネットワークから MQTT ブローカー（Mosquitto）への接続試行、API Gateway への未認証リクエスト。
    *   **判定基準:** 全ての不正アクセスが拒否され、ログに記録されること。
*   **データ暗号化:**
    *   **内容:** 通信パケット（HTTPS/MQTT over TLS）をキャプチャ。
    *   **判定基準:** プレーンテキストでデータが露出していないこと。

## 4. 安全性・フェイルセーフ (Safety)
*   **電源断テスト:**
    *   **内容:** 動作中にロボットのバッテリーを抜去、またはエッジPCの電源を強制OFF。
    *   **判定基準:** 
        1.  ロボットが慣性で動き続けず、自重や慣性ブレーキで即座に停止すること。
        2.  エッジPC側で LWT (遺言) がクラウドに届き、WebUI に「オフライン」が表示されること。
*   **近接回避速度テスト:**
    *   **内容:** 犬（ターゲット）が $1.0m/s$ でロボットに接近した際、接触前にロボットが停止できるか。
    *   **判定基準:** 物理的な接触が発生する前に、相対距離 0.3m 以上の地点で完全停止すること。
